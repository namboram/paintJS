# paintjs
Painting Board made with VanillaJS

바닐라 자바스크립트로 🎨그림판 만들기🎨
---  
+  Fill 버튼을 클릭하면 전체 채워지고 다시 Paint 버튼을 쿨락하면 선으로 그려지는 효과.
+  input:range를 이용해 브러쉬 크기 조절.
+  save 버튼을 클릭하면 png이미지 파일로 저장된다.
